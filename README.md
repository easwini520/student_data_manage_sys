# student_data_manage_sys
Student Data Management System is a simple web-based application built using HTML, CSS, and JavaScript. It allows users to manage student records with basic CRUD operations (Create, Read, Update, Delete).


✨ Features
Add Student → Enter name, roll number, and course details.

View Records → Displays all student data in a table format.

Edit Student → Update existing student details directly from the table.

Delete Student → Remove unwanted records instantly.

Persistent Storage → Uses browser localStorage to save data, so records remain even after refreshing the page.

Colorful UI → Clean design with gradient background, styled table, and colored highlights for student details (Name, Roll No, Course).
